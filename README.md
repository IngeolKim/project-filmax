FILMAX 는 영화 예매 사이트 및 관리자 페이지 입니다.

상영중/예정인 영화 목록과 정보를 제공하고 예매기능을 제공합니다.

영화별로 후기와 좋아요를 기록할 수 있습니다.

FILMAX를 통해 영화 관람을 즐겨보세요~🍿
